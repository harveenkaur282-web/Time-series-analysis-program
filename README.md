# Time-series-analysis-program

A time series analysis and forecasting project built during a course organized by the Consulting and Analytics Club, IIT Guwahati.

## Project Overview

This repository contains materials and exercises focused on time series data visualization, analysis, and forecasting using Python. The work covers both traditional statistical methods and modern machine learning approaches.

## Topics Covered

- Time data visualization in Python
- Autocorrelation and its interpretation
- Model selection and the role residuals play in evaluating model performance
- Stationarity and methods to test for its presence
- Facebook Prophet library for forecasting
- AR, MA, ARMA, and ARIMA modeling techniques
- Deep learning approaches for time series forecasting, including RNNs and LSTMs

## Structure

- `WEEK_1/`, `WEEK_2/`, `WEEK_3/`, `WEEK_4/` contain daily notebooks and exercises
- `CAPSTONE_PROJECT/` contains the final project work

## Getting Started

Run the notebooks with Jupyter or JupyterLab to explore the visualizations and models.
